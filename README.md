## 	🧰 Getting Started

<!-- Run Locally -->
### 🏃‍♀️ Run Locally

Clona el proyecto

```bash
  git clone https://github.com/mflamanna/CrochetCompany.git
```
Instala las dependencias

```bash
  npm i
```

Abre en el navegador

```bash
  npm start
```
